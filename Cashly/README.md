## üèÅ Getting Started

1.  **Clone & Install**:
    ```bash
    git clone https://github.com/2006-2006/Cashly.git
    cd Cashly
    
    # Install Server Dependencies
    cd server && npm install
    
    # Install Client Dependencies
    cd ../client && npm install
    ```
2.  **Environment Setup**:
    Configure your `.env` in the server directory with your MongoDB URI and API keys.
3.  **Launch Intelligence**:
    ```bash
    # Terminal 1 (Server)
    cd server && npm run dev
    
    # Terminal 2 (Client)
    cd client && npm run dev
    ```
